# The Open Movie DatabaseThis is a repo for managing machine readable artifacts for the [The Open Movie Database APIs](http://www.omdbapi.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [The Open Movie Database APIs](http://www.omdbapi.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.omdbapi.com/)- [Developer](http://www.omdbapi.com/)The [APIs.json](https://github.com/api-evangelist/the-open-movie-database/blob/master/apis.json) index provides links to all the relevant links for the The Open Movie Database APIs, including the OpenAPI and Postman if they are present.
